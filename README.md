## Environment Setup:

# Recommendations

- Install Expo Go on mobile device
- Develop inside a virtual environment (recommend Conda)

# Frontend Setup

```
npm install
cd .\source\frontend\
npx expo start -c
```

# Backend Setup

```
pip install -r requirements.txt
cd .\source\backend\
flask --app server_main run
```
